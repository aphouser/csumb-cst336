What language should I learn?

You've decided to take the plunge into programming and want to learn a language, but you don't know where to start? Take a look at these suggestions and maybe that'll help you make a decision.

If you have the desire to build apps for Android or iPhone, take a look at our Mobile section. Here you can find some information on Swift and Java primarily.

Maybe web design is calling your name. Look into the Web section to learn more about HTML, CSS, and JavaScript for frontend, and PHP for backend.

Lastly, if you're just after some good old object oriented programming, take a look in the General Purpose area for info on Java and Python as some starters.

If you're still on the fence, do a quick search on why you may want to learn. Or, let us help you out with a link. Here you can see thoughts that coding helps you think and learn to solve problems and it can make things easier for you.
